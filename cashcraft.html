À propos du projet

Cashcraft est une plateforme e-commerce complète conçue pour faciliter les transactions en ligne. Elle offre une expérience d'achat fluide avec un panier d'achat intégré, des options de paiement sécurisées, et un système de gestion des commandes pour les vendeurs. Développée avec React pour une interface utilisateur dynamique et interactive, et Node.js pour un back-end robuste, cette plateforme intègre des API de paiement comme Stripe et PayPal. Les fonctionnalités incluent la recherche avancée de produits, les avis clients, et un tableau de bord pour les vendeurs.

Fonctionnalités principales

Panier d'achat

Système de panier persistant avec calcul automatique.

Paiements sécurisés

Intégration Stripe et PayPal pour transactions sûres.

Gestion vendeurs

Tableau de bord pour suivi ventes et inventaires.

Avis clients

Système de notation et commentaires utilisateurs.

Technologies utilisées

React Node.js MongoDB Stripe Tailwind CSS
