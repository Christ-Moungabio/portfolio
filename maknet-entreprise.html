À propos du projet

Maknet Entreprise est une plateforme de gestion complète développée pour faciliter la gestion des activités d'une entreprise. Elle permet de gérer les inscriptions des utilisateurs, les paiements en ligne, et offre un tableau de bord intuitif pour les administrateurs. Développée avec Laravel pour le back-end robuste et sécurisé, et React pour une interface utilisateur moderne et réactive, cette plateforme assure une expérience fluide pour tous les utilisateurs.

Fonctionnalités principales

Gestion inscriptions

Système complet de gestion des inscriptions utilisateurs.

Paiements intégrés

Paiements en ligne sécurisés via API tierces.

Tableau de bord

Interface intuitive pour administrateurs.

Rapports détaillés

Analyse des performances avec rapports complets.
